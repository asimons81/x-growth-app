# X Growth App (Project X)

## 🎯 Mission
Build a tool to help Tony grow on X (Twitter) by analyzing top-performing content and providing actionable insights.

## 🛠️ Stack
- **Framework:** Next.js 15 (App Router)
- **Database:** Supabase (PostgreSQL)
- **Styling:** Tailwind CSS + shadcn/ui
- **Auth:** Supabase Auth
- **Deployment:** Vercel

## 📅 Roadmap

### Phase 1: Foundation (Current)
- [x] Next.js setup
- [x] Supabase integration
- [ ] Authentication (Twitter/X OAuth)
- [ ] Basic Dashboard UI

### Phase 2: Analytics
- [ ] Fetch user metrics
- [ ] Track follower growth

### Phase 3: AI Insights
- [ ] Content suggestions
